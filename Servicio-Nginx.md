# Servicio NGINX
---
#### Instalar Nginx
```sh
$ sudo apt-get install nginx
```

#### Saber el estado de Nginx
```sh
$ service nginx status
```
#### Iniciar el servicio de Nginx
```sh
$ service nginx start
```

#### Detener el servicio de Nginx
```sh
$ service nginx stop
```

#### Reiniciar el servicio de Nginx
```sh
$ service nginx restart
```
